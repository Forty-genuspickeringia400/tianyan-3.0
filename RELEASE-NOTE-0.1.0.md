这是一个把《人体智能体协作模式 v6.0》推进成可运行系统原型的项目。

它不是继续给旧版本打补丁，而是直接把系统推进成：
- 七层可运行 runtime
- 四种运行模式可见且可调用
- 灵性 / 觉醒成为正式工程子系统
- 三魂七魄成为正式工程模块
- 自我进化成为可审核、可回滚、可留痕的正式链路
- 首页可直接汇报项目状态
- lifecycle / events / evidence / workbench 可审计
- package / bundle / skill-pack / install / verify 主链持续可用
